                                           3.Tilde Expansion
                                       ===============================
------------------------------------------------------------------
1.Single tilde `~`:  
      
      ```bash
      echo ~
      ```
      Output:
      ```
      /home/username
      ```
------------------------------------------------------------------
2.Tilde followed by a username `~username`:  
      
      ```bash
      echo ~root
      ```
      Output:
      ```
      /root
      ```
------------------------------------------------------------------
3.Tilde followed by a path `~/path`:  

      ```bash
      echo ~/Documents
      ```
      Output:
      ```
      /home/username/Documents
      ```
------------------------------------------------------------------
4.Tilde followed by a username and path `~username/path`:  
      
      ```bash
      echo ~john/Documents
      ```
      Output:
      ```
      /home/john/Documents
      ```
------------------------------------------------------------------
