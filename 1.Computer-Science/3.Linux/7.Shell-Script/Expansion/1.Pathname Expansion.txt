______________________________________________________________________________________________________________________________________
                                       1.Pathname Expansion (Globbing)
                                       ===============================
                                            For Read-Only Commands
                                            ======================
__________________________________________________________________________________
1.Asterisk `*`
       
                    ls *.txt
                    ls file*
                    ls *s*     
                    ls s*u*                   -> success
      
__________________________________________________________________________________
2.Question Mark `?`
      
                    ls file?.txt
                    ls file??.txt
                    ls ?file?.txt
      

__________________________________________________________________________________
3.Brackets `[ ]`
      
                    ls file[1-4].txt
                    ls file[!1-4].txt
                    ls file[?1-4].txt
                    ls file[a-z].txt
                    ls file[a-z][A-Z].txt
      
__________________________________________________________________________________
4.Escaping Special Characters:
      
                    ls file\*.txt
      
__________________________________________________________________________________
5.Hidden Files and Globbing:
      
                    ls .*

__________________________________________________________________________________
6.Compination:
      
                    ls file*[1-3].txt
      
__________________________________________________________________________________
7.Disabling Pathname Expansion:
                  
                    set -f              disable
                    set +f              enable
