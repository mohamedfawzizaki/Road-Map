                                                         sftp
                                                     ===============
 __________________________________________________________________________________________________________________________________
|       Option      |          Example        |                      Description                                                                                  
|___________________|_________________________|____________________________________________________________________________________
| sftp              | sftp user@host          | Starts an SFTP session to a remote system.                                                        
|-------------------|-------------------------|------------------------------------------------------------------------------------ 
| get               | get file.txt            | Downloads a file from the remote server to the local machine.                                      
| put               | put file.txt            | Uploads a file from the local machine to the remote server.                                       
| mget              | mget .txt               | Downloads multiple files matching the pattern (e.g., .txt) from the remote server.            
| mput              | mput .txt               | Uploads multiple files matching the pattern (e.g., .txt) to the remote server.                
|___________________|_________________________|____________________________________________________________________________________
